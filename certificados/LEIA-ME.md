# Pasta de Certificados

Coloque aqui os arquivos dos seus certificados (imagem ou PDF) e eles
aparecerão automaticamente no carrossel da seção "Certificados e cursos"
do site — sem precisar mexer em nenhum código.

## Como usar

1. Exporte/tire print do certificado como imagem (`.png`, `.jpg`, `.jpeg`,
   `.webp`, `.gif`) ou salve como `.pdf`.
2. Faça upload do arquivo diretamente nesta pasta `certificados/` no
   repositório do GitHub (pelo site do GitHub: **Add file → Upload files**,
   ou via `git add / commit / push`).
3. Pronto! Em alguns segundos após o GitHub Pages atualizar, o certificado
   já aparece no carrossel do site.

## Dicas

- O nome do arquivo vira o título do card. Use nomes descritivos, por
  exemplo: `google-it-support.png` ou `git-e-github-udemy.pdf`.
- Para remover um certificado do site, basta apagar o arquivo desta pasta.
- Este arquivo `LEIA-ME.md` é só instrução e é ignorado pelo carrossel —
  pode deixá-lo aqui sem problema.
