## English

### Overview

A professional personal portfolio with projects automatically loaded from the GitHub API. This minimalist, single-file portfolio showcases your work and is hosted for free on GitHub Pages.

### Project Structure

```
index.html              Single-page application (HTML + CSS + JavaScript)
thaynan-foto.png        Profile picture
THAYNANCurriculo.pdf    Resume/CV for download
```

### Getting Started with GitHub Pages

1. Create a repository named `thaynan-passos.github.io` on GitHub
2. Upload the 3 files from this portfolio
3. Go to **Settings → Pages** and set **Branch: main / root**
4. Save — your portfolio will be live in approximately 1 minute at: `https://thaynan-passos.github.io`

### Project Updates

The "Projects" section automatically fetches your public repositories from the GitHub API whenever someone visits your site. No manual updates needed — everything syncs automatically.

## 日本語

### 概要

GitHub APIから自動的にプロジェクトを読み込むプロフェッショナルな個人ポートフォリオです。このミニマリストな単一ファイルポートフォリオはあなたの仕事を紹介し、GitHub Pagesで無料でホストされます。

### プロジェクト構造

```
index.html              シングルページアプリケーション (HTML + CSS + JavaScript)
thaynan-foto.png        プロフィール写真
THAYNANCurriculo.pdf    ダウンロード用の履歴書
```

### GitHub Pagesでの公開方法

1. GitHubで`thaynan-passos.github.io`という名前のリポジトリを作成します
2. このポートフォリオの3つのファイルをアップロードします
3. **Settings → Pages**に移動し、**Branch: main / root**に設定します

「プロジェクト」セクションは、誰かがサイトを訪問するたびにGitHub APIから自動的にあなたの公開リポジトリを取得します。手動更新は不要です — すべて自動的に同期されます。
