# Portfólio — Thaynan Passos

Portfólio pessoal com projetos carregados automaticamente via GitHub API.

## Estrutura

```
index.html           ← tudo em um arquivo só (HTML + CSS + JS)
thaynan-foto.png     ← foto de perfil
THAYNANCurriculo.pdf ← currículo para download
```

## Como hospedar no GitHub Pages (grátis)

1. Crie um repositório chamado `thaynan-passos.github.io` no GitHub
2. Faça upload dos 3 arquivos desse portfólio
3. Acesse **Settings → Pages → Branch: main / root**
4. Salve — em 1 minuto o site estará em: `https://thaynan-passos.github.io`

## Atualização dos projetos

A seção "Projetos" busca seus repositórios públicos direto da API do GitHub sempre que alguém abre o site — não precisa editar nada, atualiza sozinho.
